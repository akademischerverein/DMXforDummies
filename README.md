# Benötigte Bibliotheken

DMXforDummies benötigt die Bibliotheken DmxLib und DmxLib.Util, welche aus dem Repo [Jonathan/dmx-lib](https://gitlab.jensgutermuth.de/jonathan/dmx-lib) gebaut werden können. Zur Vereinfachung werden die aktuellen Release-Builds dieser Bibliotheken ins Root-Verzeichnis des DMXforDummies-Repo gepackt und zum Bauen verwendet.
